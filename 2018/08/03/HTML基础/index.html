HTML基础

html全称是 Hyper Text Markup Language.（超文本标记语言）
1.随记
.除了div和span，其他的标签都有默认的样式。
.strong是一个逻辑状态，bold是一个物理状态
.html没有块级元素和内联元素的，css负责样式
.body 标签的默认 margin 是8像素

2.空标签
空标签即不能包含子元素的标签。
如：input;img;meta;link;hr;br;col;colgroup;frame;base;等。

3.可替换元素
可替换元素（replaced element）的展现不是由CSS来控制的。这些元素是一类 外观渲染独立于CSS的 外部对象。 典型的可替换元素有 img、object、video 和 表单元素，如textarea、 input 。 某些元素只在一些特殊情况下表现为可替换元素，例如 audio>和 canvas 。 通过 CSS content 属性来插入的对象 被称作 匿名可替换元素

4.块级元素和行内元素以及display:block
根据CSS规范的规定，每一个网页元素都有一个display属性，用于确定该元素的类型，每一个元素都有默认的display属性值，比如div元素，它的默认display属性值为“block”，成为“块级”元素(block-level)；而span元素的默认display属性值为“inline”，称为“行内”元素。

块级元素：动占据一定矩形空间，可以通过设置高度、宽度、内外边距等属性，来调整的这个矩形的样子；

行内元素：自己的独立空间，它是依附于其他块级元素存在的，因此，对行内元素设置高度、宽度、内外边距等属性，都是无效的。
css布局 ：横向布局 /纵向布局

5 iframe 标签：
嵌套页面 （使用iframe页面会很卡不建议使用）
frameborder=”0”去除边框,src可以写相对路径。

6 iframe标签与a标签的结合使用

1.target=”_blank” 在空页面，新的页面打开链接
2.target=”_top” 在顶级窗口打开
3.target=”_self” 在当前页面打开
4.target=”_parent” 在父级页面打开

7.a 标签
跳转页面 （HTTP GET请求）get默认会把参数放到查询参数里面

1.无协议写法 （ #号不发起请求）
2.相对路径 只引目录为参考
3.伪协议 希望这个a 点击后什么都不做的标签

8 form 标签
跳转页面 （HTTP POST请求）post默认会把参数放到第四部分里面


1.必须要有submit 提交按钮，否则无法提交
2.name 出现在四部分
action可以修改为查询参数。

9.input/button
input是没有子元素，button是有子元素的。

submit是唯一能决定这个form表单能不能点击提交的按钮，如果type是button，只是普通按钮。

如果form中只有一个《button》button《/button》，那么button标签会自动升级为submit提交按钮。

10.checkbox 复选框 radio 单选框

10.select 下拉选框

multiple 可以多选
disabled 不让选
selected 默认选择

11.textarea 文本框
可以用css改变宽高

12.table （标签顺序没有影响）
th 表头
td 数据
col (colgroup) 列

table-collpase: collpase; 可以消除表格内框
